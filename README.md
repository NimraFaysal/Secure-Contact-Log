# Secure-Contact-Log
**Secure Contact Log** is a secure application used to store, manage, and search contact information safely. It ensures data privacy through authentication and a layered architecture that improves security and maintainability.
A Contact Management System (CMS) is designed to help users efficiently store, organize, and manage contact information like names, phone numbers, email addresses, and other personal details. 
HIGHLIGHTS OF MY SCOPE:

Storing Contact Information:
Users can easily add, store, and view important contact details such as names, phone numbers, and email addresses.
You can add custom fields (e.g., address, date of birth, etc.) to make the contact information more detailed.

Managing and Organizing Contacts:
Users can search, update, or delete contacts.
The system allows sorting of contacts by name, phone number, or other fields to quickly find the needed information.

Efficient Searching:
Users can search for contacts by different parameters like name, phone number, or email. This saves time in locating specific contacts from a large list.
